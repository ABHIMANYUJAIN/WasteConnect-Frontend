# WasteConnect Frontend

Frontend for WasteConnect, a smart waste management and logistics platform.

## Features

- JWT Authentication
- Role-Based Access Control
- Admin Dashboard
- Collector Dashboard
- User Dashboard
- Pickup Request Management
- Route Optimization
- Analytics & Leaderboards

## Tech Stack

- React
- Vite
- Tailwind CSS
- Axios
- React Router

## Run Locally

npm install
npm run dev
